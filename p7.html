HTML

HTML son las siglas del Hyper Text Markup Language, que puede traducirse como lenguaje de marcas o marcado de hipertexto.

El lenguaje HTML se emplea para crear páginas web.Es muy fácil ver el código HTML de una página web, la opción exacta cambia de un navegador a otro y tambien puede cambiar de una versión a otra de un msimo navegador, pero suelen tener un nombre similar

HTML se compone de etiquetas que se escriben entre los simbolos menor y mayor que.
